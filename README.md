# Careerist QA Automation Coursework

This repository contains my coursework and assignments from the **Careerist QA Automation Engineering program**.

The exercises in this repository were completed as part of structured training designed to teach the fundamentals of **software testing, automation frameworks, and browser automation**.

---

# Technologies Used

| Technology | Purpose |
|------------|--------|
| Python 3 | Core programming language |
| Selenium | Browser automation |
| Behave | Behavior-Driven Development (BDD) testing framework |
| Gherkin | Human-readable test scenarios |
| PyTest | Test execution and validation |

# What This Repository Demonstrates

This coursework demonstrates practical experience with:

- Writing automated tests using **Python**
- Browser automation with **Selenium WebDriver**
- Behavior-Driven Development using **Behave**
- Writing **Gherkin feature files**
- Implementing **step definitions**
- Basic **test framework structure**
- Automated test execution
